1. Run the below commands for setting up the environment:

conda create -n env_name python=3.9
conda activate env_name
cd diffusers
pip install -e .
cd ..
pip install -r requirements.typing_extensions

2. Install ffmpeg using the below command:

conda install menpo::ffmpeg

3. Now when you try to run any code that involves importing scikit-video,
you will encounter the attributeError: "module 'numpy' has no attribute 'float'."

4. Go to the ./anaconda3/envs/env_name/lib/python3.9/site-packages/skvideo/io/abstract.py,
and change np.float to float and also change np.int to int

5. Now you are good to go.
